## 📖 About The Project

This portfolio website is designed to present:

- Personal introduction
- Services offered
- Portfolio projects
- Client testimonials
- Blog section
- Contact form

The goal of this project was to practice building a complete multi-section responsive layout using only HTML and CSS.

---

## ✨ Features

- 🏠 Hero Section with background image
